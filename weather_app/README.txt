Weather App Sederhana
=====================

Isi:
- index.html
- style.css
- script.js

Fitur:
- Ambil data cuaca dari OpenWeatherMap
- Animasi loading saat fetch
- Deteksi lokasi otomatis (GPS)
- Fade-in transition saat hasil muncul
- Responsive

Cara pakai:
1. Ekstrak zip
2. Buka file index.html di browser
3. Pastikan koneksi internet aktif
